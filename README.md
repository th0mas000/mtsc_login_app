# MagicTech - MTSC Login App

```bash
# 1. Clone โปรเจกต์จาก GitHub
git clone https://github.com/th0mas000/magictech.git

# 2. เข้าไปที่โฟลเดอร์โปรเจกต์หลัก
cd magictech

# 3. เข้าไปที่โฟลเดอร์แอป mtsc_login_app
cd mtsc_login_app

# 4. ติดตั้ง dependencies ที่จำเป็น
flutter pub get

# 5. รันแอป
flutter run
```
